Git es un software de control de versiones, mediante el cual tenemos un repositorio local, quiere decir que está en una máquina, no se necesita conexión a internet a menos que se desee sincronizarlo y tenemos también un repositorio remoto, al cual podemos acceder desde internet, sin importar que no estemos en la máquina. Por medio de este control de versiones podemos iniciar el fascinante mundo del diseño de páginas web, ya que es uno de los más usados y tiene bastantes herramientas para explorar.
